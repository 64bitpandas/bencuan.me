## typography

Headers are set in <span style={{ fontFamily: 'Pangaia, sans-serif' }}>Pangaia</span>, body text is set in <span style={{ fontFamily: 'Eiko, serif' }}>Eiko</span>, and UI elements are set in <span style={{ fontFamily: 'Fraktion, sans-serif' }}>fraktion sans</span>.

Sizing is calculated with a modified version of [Fluid Typography](https://fluidtypography.com/).

All fonts are designed by [Pangram Pangram](https://pangrampangram.com), and licensed for personal use.
